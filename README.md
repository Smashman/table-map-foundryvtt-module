# TableMap - Display FoundryVTT upon your table with embedded screen!

## Requirements

-   Node
-   NPM

## Set-up

1. `npm i`
2. `npm run build`
3. Symlink the `dist` folder into `FoundryVTT/Data/modules/table-map`
